## New findings (conservative)
The retained operator ladder is:

```text
AppInit_DLLs configured
  != mechanism enabled (LoadAppInit_DLLs)
  != eligible under signature gate (RequireSignedAppInit_DLLs)
  != (Win8+ Secure Boot) mechanism not disabled
  != loaded into one concrete interactive process
  != later effect owned by that DLL
```

Two practical discriminants preserved:
- **Secure Boot on Win8+** can make the whole mechanism dead even if artifacts exist
- **signature gate posture** can make an unsigned payload non-loadable (do not assume the registry write implies execution)
